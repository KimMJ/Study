# Cache

## Cache란?

* 자주 사용되는 데이터를 빠르게 가져올 수 있도록 해주는 임시 저장소
* 데이터가 필요할 때 빠르게 가져와서 사용하기 위해 잠시 데이터를 저장해 두는 곳.

## 웹 서비스에서의 Cache

사용자에게 빠른 응답을 제공해 주기 위해 사용하는 임시 저장소

### 대상

1. 웹페이지 결과
2. 복잡한 계산을 통해 얻어지는 객체
3. DB로부터 조회한 결과
4. 기타

### 종류

1. Local Cache
2. Remote Cache

## Local Cache vs Remote Cache

### Local Cache

* 어플리케이션 내부에서 사용하는 캐시
* 빠름
* 다른 어플리케이션과 데이터 공유에 어려움이 있음.
* 다른 어플리케이션이 가진 값을 동기화하는데 어려움이 있음
* 로컬의 자원 사용

### Remote Cache

* 여러 어플리케이션에 공유를 목적으로 사용.
* Local Cache 보다 느리다.
* 데이터를 물리적으로 분산 저장할 수 있다.
* 대용량
* memcached, ARCUS
